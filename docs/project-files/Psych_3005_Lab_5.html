Create New Variable / Column (‘POORHLTH_BINS’) to group data

if “POORHLTH” == 0, then “POORHLTH_BINS” == 0, if “POORHLTH” = 1 - 7, then “POORHLTH_BINS” == 1, if “POORHLTH” = 8 - 14, then “POORHLTH_BINS” == 2, if “POORHLTH” = 15 - 21, then “POORHLTH_BINS” == 3, if “POORHLTH” = 22 - 30, then “POORHLTH_BINS” == 4

conditions = [df['POORHLTH'] == 0, (df['POORHLTH'] >= 1) & (df['POORHLTH'] <= 7), (df['POORHLTH'] >= 8) & (df['POORHLTH'] <= 14), (df['POORHLTH'] >= 15) & (df['POORHLTH'] <= 21), (df['POORHLTH'] >= 22) & (df['POORHLTH'] <= 30)]
values = [0, 1, 2, 3, 4]
df['POORHLTH_BINS'] = np.select(conditions, values, default = np.nan) # np.select(conditions, values) --> conditions = list of conditions, values = list of values to choose from
